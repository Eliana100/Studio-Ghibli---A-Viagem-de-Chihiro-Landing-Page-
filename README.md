*Projeto no Figma:* https://www.figma.com/design/GigjlmVOEl5yjPArNJFfhQ/Desafios---CodeLab--Copy-?node-id=5854-2

<br>

## 🌸 Desafios

<br>

Neste projeto, o foco foi dominar a organização de elementos em tela e a criação de componentes interativos:

- **CSS Grid Layout:** Utilizado para estruturar a página principal, dividindo o conteúdo textual da área visual de forma precisa.
- **Flexbox:** Aplicado para o alinhamento fino de itens internos, como a navegação de ícones sociais e a centralização do conteúdo dentro dos botões.
- **Manipulação do DOM:** Implementação de um Modal interativo em JavaScript para exibição do trailer do filme sem sair da página.
- **Variáveis CSS:** Uso de Custom Properties para gerenciar a paleta de cores e fontes.
